## Plain Text
Normal text content is just that, normal text content. It is displayed as it is.
```html
I code, therefore I am
```
output:
>I code, therefore I am


## Comments
To make comments, use "<--!   -->"

```html
<!-- This is a comment -->
```

## Content

- ***Headers***: There are different levels of headers, from biggest to smallest, from h1 to h6
- ***Paragraph***: To create a block level, separated block of text
- ***Image***: To insert image
- ***Button***: To create a clickable component
- ***Input***: To create an input field 
- ***Anchor Tags***: To link to another page
- ***List***: To create lists, ordered and unordered
