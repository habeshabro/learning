## Learning

💡There is an existing model, data is introduced to it, it changes the parameters in the model to adjust to the data. This process is called **learning** . A model is simply a process used to generate data

## Utility Function
The tool we use to compare the generated data to actual data is called the utility function, i.e. it is the way we judge how well a model is doing
