use double square brackets to make a link
[[MD Language]]
use a | in the middle to change the name visible
[[MD Language|Obsidian]]

use # to make headings, more # , lower heading type
## Heading one
### Heading two
#### Heading three
##### Heading four


new line between sentences causes creation of a paragraph

use double asterisks (\*\*) to bold something like thing **bold**
single asterisk(\*) causes italics, like so *italic*

to create a block quote, use the greater than symbol like so
>alas my love
>you do me wrong
>to cast me away so discourteously


use the minus symbol (-) to make a list, indent to make sub list
- item one
	- item two
- etc...

use number followed by period to make a number list
1. something
2. something else
