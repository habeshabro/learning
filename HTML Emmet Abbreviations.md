 To generate random text lorem+no:
 ```html
 lorem50
 ```
 to nest elements use >:
 ```html
 p>lorem50
 ```
 this produces 50 random words inside of a paragraph

"+" is used to add elements
"\*" is used to create multiple elements
```html
div+p*3

<div></div>
<p></p>
<p></p>
<p></p>
```
